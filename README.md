# amicae: Your Campus Connection Companion
![amicae_screens](https://github.com/user-attachments/assets/1ffe3b51-6304-4f81-8492-cc79dd47fdc3)

## Project Overview

Amicae is a mobile application built with Flutter to help Concordia University students connect with study buddies, mentors, campus events, and study spotsâ€”all in one place. The goal is to reduce academic isolation and make campus life more connected, supportive, and engaging. 

## Key Features

- **Swipe and Study**: Match with students and mentors based on shared interests, courses, or study habits.
- **Amicae MatchUp AI**: AI-powered recommendations for study partners and mentors using Vertex AI.  
- **Study Spot Finder**: Discover nearby study-friendly locations around downtown Montreal.  
- **Events Promotion**: Stay informed about upcoming university events and opportunities.  

## Technologies Used

- **Flutter** - Cross-platform mobile framework for iOS and Android using Dart  
- **Firebase** - Back-end services including authentication and real-time database  
- **Vertex AI** - AI integration for personalized match suggestions
- **Concordia Open Data API** - Source for campus events and information.
